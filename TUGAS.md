## Buatlah project Laravel dengan ketentuan sebagai berikut :

1. Menggunakan Laravel framework, dikerjakan secara berkelompok, dan harus ada satu ketua

2. Dalam bentuk sistem informasi dengan tema bebas (harus bisa CRUD database)

3. Setiap anggota wajib commit melalui Github (buat satu repository untuk dikerjakan secara bersama - kolaborasi)

4. Project live dan bisa diakses secara online. Gunakan Laravel Cloud dan online db. Boleh menggunakan alternative lain seperti fly io dan render untuk deploy. Supabase dan neon untuk database.
