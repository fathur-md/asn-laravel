## User Flow

```
Guest
  ├── Visit Home → See stats, featured campaigns, categories
  ├── Browse /campaigns → Filter by category, see progress
  ├── View /campaigns/{id} → Read detail, see donors & comments
  ├── Register → Create account (/register)
  └── Login → Access dashboard (/login)

Authenticated User
  ├── Dashboard → See my campaigns & recent donations
  ├── /campaigns/create → Create new campaign
  ├── /campaigns/{id}/edit → Edit own campaign
  ├── /campaigns/{id}/delete → Delete own campaign
  ├── /campaigns/{id}/donate → Donate to any active campaign
  ├── /campaigns/{id}/comment → Comment on any campaign
  └── Logout → Redirect to home
```
